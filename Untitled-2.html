🔍 Linear Search O(n)

Pencarian berurutan · Data TIDAK perlu terurut · Tanpa batas data

📋 Contoh data besar (25+ angka)
🎲 Generate data acak (100 angka)
💡 Hasil pencarian akan tampil di sini
Linear search akan menyisir setiap elemen dari indeks 0 hingga akhir.
📌 Prinsip: Membandingkan tiap elemen satu per satu → cocok → return indeks.
✔ Kelebihan: data boleh acak / tidak terurut, tanpa batasan jumlah. ✖ Kekurangan: O(n) untuk data besar.

⚡ Binary Search O(log n)

Pencarian bagi dua · WAJIB data terurut (akan diurutkan otomatis)

📋 Contoh data besar (acak)
🎲 Generate data acak (100 angka)
💡 Hasil pencarian + indeks setelah sorting
Binary search hanya bekerja pada data terurut. Array akan diurutkan ascending terlebih dahulu.
⚠️ Syarat mutlak: data harus TERURUT (ascending). Program akan mengurutkan otomatis, lalu mencari dengan O(log n).
🖥️ Implementasi algoritma berdasarkan kode C++ pada modul (Codeblocks).
Linear Search menggunakan loop & flag boolean. Binary Search menggunakan prinsip bagi dua (low, mid, high).
